Annotation Interface Instructions


    This task consists of 3 stages

  • Identification of distinct sounds
  • Localization of these sounds
  • Matching each location with its corresponding sound

    • We expect this task to take around 10 minutes of your time.
    You will be paid $2.5 for completing the task.

Page 1 / 7

Annotation Interface Instructions

The first stage is to identify the number of distinct sounds.

  • Press the Play button to listen to the audio recording.
  • While listening, count the number of distinct sounds that you heard.
  • You can use the text box to take notes about the sounds you hear and their locations. It might help you when localizing the sounds.
  • You may listen to the audio as many times as you want.

Page 3 / 7

Annotation Interface Instructions

  • After listening to the full audio recording, select the number of distinct sounds you heard using the dropdown menu. Remember the only types of sounds in the recording are those listed in page 2.
  • You may change this number at any time.

Page 4 / 7

Annotation Interface Instructions

The second stage is to localize these sounds.

  • After selecting the number of sounds, you need to indicate the spatial location of each sound that you heard. We will refer to these spatial locations as annotations.
  • In this stage you will see 4 spatial displays: a 2D "top view" display on which you can annotate the horizontal spatial location; a 2D “back view” display on which you can annotate the vertical spatial location if a sound is to either side of you; a 2D “side view” display on which you can annotate the vertical spatial location if a sound is in front or behind you; and a rotatable 3D view which is just for displaying the spatial location annotations you create.

Page 5 / 7

Annotation Interface Instructions

  • To add a new annotation press key Option or Alt and click on the desired 2D view.
  • To remove an annotation press key Control or Ctrl and click on the desired annotation.
  • You can always see the rules in the "Keyboard Rules" button.
  • To annotate the spatial location of each sound, you must select a horizontal spatial location and a vertical spatial location.

Page 6 / 7

Annotation Interface Instructions

  • If you need to adjust the location of your annotation, you may click on the desired point and move it, or use the + and - buttons.
  • Click submit once you have completed your annotation.

Page 7 / 7

Annotation Interface Instructions

  • In the last stage of this task, match each location you annotated with its corresponding sound.
  • The set of sounds available for matching consists of all of the sounds that were present in the recording.
  • Please listen to all the sounds and match them to the annotated spatial locations.
  • Each sound can only be matched once.

You will go through a practice round first. You can repeat the practice round if needed.
